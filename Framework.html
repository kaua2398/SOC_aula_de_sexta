Diretrizes do NIST - Cybersecurity Framework

  • Identificar
  • Proteger
  • Detectar
  • Responder
  • Recuperar

Identificar

O primeiro passo é entender os ativos, sistemas e riscos para a segurança da informação.

Exemplo: Realizar uma análise de risco para mapear vulnerabilidades.

Proteger

Implementar salvaguardas para garantir a proteção dos ativos e dados.

Exemplo: Aplicar autenticação multifator em sistemas críticos.

Detectar

Monitoramento contínuo para identificar atividades suspeitas ou maliciosas.

Exemplo: Uso de SIEM para detectar acessos não autorizados.

Responder

Tomada de ações rápidas para mitigar impactos de incidentes de segurança.

Exemplo: Ativar um plano de resposta a incidentes para conter um ataque.

Recuperar

Restaurar a operação normal e melhorar as estratégias de segurança.

Exemplo: Restaurar backups seguros após um ataque de ransomware.
